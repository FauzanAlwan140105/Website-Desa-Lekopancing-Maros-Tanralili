# Website-Desa-Lekopancing-Maros-Tanralili

Website Desa Lekopancing merupakan website sistem informasi desa yang dibuat untuk menyediakan informasi resmi mengenai Desa Lekopancing yang berada di Kecamatan Tanralili, Kabupaten Maros. Website ini bertujuan untuk membantu masyarakat dalam mengakses informasi desa secara digital dengan mudah dan cepat.

📌 Tujuan Project

Project ini dibuat untuk:

Menyediakan informasi resmi tentang Desa Lekopancing

Menyampaikan berita dan kegiatan desa

Mendukung digitalisasi pelayanan desa

Mempermudah masyarakat dalam mengakses informasi desa
